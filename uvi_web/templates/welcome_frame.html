Welcome to the Unified Verb Index

hosted at the University of Colorado Boulder

The Unified Verb Index is a system which merges links and web pages from four different natural language processing projects:


Representing 9344 verbs in:
6791 VerbNet senses, 4577 VerbNet lemmas, 329 VerbNet main classes, 272 VerbNet subclasses, 5649 PropBank links, 4186 FrameNet links, and 4898 Grouping links
